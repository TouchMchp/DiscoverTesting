# Firmware
The firmware facilitates configuring the ATtiny817 QTouch Moisture Demo kit. Here, tThe MPLAB Data Visualizer utilizes the graphical user interface (GUI) to display the touch state and Delta.

The precompiled binary for the AYiny817 is available in the [bin](../bin) directory.  

# MPLAB Data Visualizer Configuration

Refer https://developerhelp.microchip.com/xwiki/bin/view/applications/touch-gesture/mplab-data-visualizer-touch-plugin/visualize-touch-data-using-mplab-data-visualizer/ to connect MPLAB Data Visualizer and enter the below settings

- Baud Rate:38400
- [Datasteamer file location](attiny817_qtmoistd_kit.X/mcc_generated_files/touch/datastreamer)


# Recommended Versions
- [MPLAB Data Visualizer v1.3.1677](https://www.microchip.com/en-us/tools-resources/debug/mplab-data-visualizer)
- [MPLAB X IDE v6.20](https://www.microchip.com/mplab/mplab-x-ide)
